# 深夜の廻しスマホ 取扱説明書

公開URL: https://nomi-party.yukinooooooosan.cc/

## これはなに？

`深夜の廻しスマホ` は、1台のスマホを参加者で順番に回して遊ぶ、飲み会向けのパーティゲーム集です。

全員が自分の端末を出す必要はありません。最初に参加メンバーを登録し、ゲームを選んだら、画面の指示どおりにスマホを次の人へ渡して進めます。

## 大事な約束

- パスあり
- 無理強いなし
- 記録しない
- 秘密入力や秘密回答の画面は、本人だけが見る
- 結果が出たあとでも、やりたくないことは断ってOK

少し攻めた内容のゲームもありますが、場を壊さず、笑える範囲で遊ぶためのアプリです。

## 基本の遊び方

1. ページを開く
2. 参加メンバーを登録する
3. ゲーム一覧から遊びたいゲームを選ぶ
4. 画面に表示された人へスマホを渡す
5. 本人だけが入力・回答する
6. 全員分が終わったら、結果を全員で見る

メニューへ戻りたいときは、各ゲーム画面の `Menu` ボタンを押します。

## メンバー設定

最初に `今夜の顔ぶれ` 画面で参加メンバーを作ります。

- 参加人数は2人から12人まで
- 名前は空でもOK。開始時に `Player 1` のような仮名で補完されます
- `＋ メンバーを追加` で人数を増やせます
- `−` でメンバーを削除できます
- `♂` / `♀` ボタンを押すと性別表示を切り替えられます
- `このメンバーで始める` を押すとゲーム一覧へ進みます

一度遊ぶと、次回以降は `前回のメンバーを読み込む` から同じメンバーを呼び出せます。

## ゲーム一覧

現在遊べるゲームは4つです。

### 01 色あて

お題に合う色を、RGBスライダーで直感的に作って当てるゲームです。

流れ:

1. 親がランダムに選ばれます
2. 親だけがスマホを見て、今回のお題の色を作ります
3. 親以外の人が順番にスマホを受け取り、親が作りそうな色を予想します
4. 全員の回答後、親の色に近かった順に結果が出ます

遊ぶときのポイント:

- 親の入力画面は親だけが見てください
- 回答者は前の人の回答を見ずに、自分の直感で色を作ります
- 結果画面では、親の色と回答色、スコア、順位が表示されます
- `判定` ボタンで色の近さの計算方法を切り替えられます

### 02 欲望ガチャ

男性が匿名で欲望を書き、女性が「応じてもいい」と思うものだけに投票する、二重抽選ゲームです。

流れ:

1. 男性が順番に、1人1つ欲望を入力します
2. 入力内容は確定後すぐに隠れます
3. 女性が順番に、欲望リストから応じてもいいものを選びます
4. 誰がどれに投票したかは表示されません
5. まず実施する欲望を抽選します
6. 次に、その欲望に応じてくれる女性を発表します
7. 最後に、その欲望を書いた男性を発表します

遊ぶときのポイント:

- 男性と女性がそれぞれ1人以上必要です
- 男性は、相手が断れる内容だけを書いてください
- 女性は、少しでも迷うものは選ばなくてOKです
- 投票した女性全員が表示されるわけではなく、当選者だけが発表されます
- 候補者がいない欲望は `今回はいません` として不成立になります
- 成立後でもパスOKです

### 03 性癖マッチング

2択質問への回答から、噛み合うペアを1組ずつ発表する診断ゲームです。

流れ:

1. 質問数を `10問` または `20問` から選びます
2. ペアルールを選びます
3. 質問ジャンルごとの混ぜ具合を調整します
4. 参加者が順番にスマホを受け取り、本人だけで回答します
5. 全員が回答したら、相性の高いペアから順に発表されます

設定項目:

- `質問数`: 10問 / 20問
- `ペアルール`: 同性ペアも含める / 異性ペアだけにする
- `混ぜ具合`: 各ジャンルをどれくらい出すか選べます

遊ぶときのポイント:

- 回答中の画面は本人だけが見てください
- 各質問は必ずどちらかを選びます
- 結果は上位ペアから確定し、使われた人を抜いて次のペアを決めます
- 人数が奇数の場合など、ペアにならない人が出ることがあります

### 04 シンプルルーレット

参加者の名前を使って、ランダム抽選・ランダム順・ランダムペアを作る便利ツールです。

使える機能:

- `ルーレット`: 1人をランダムに選びます
- `リストソート`: 参加者をランダムな順番に並べます
- `強制ペア`: 参加者をランダムにペア分けします

ルーレットとリストソートでは、対象を選べます。

- `全員`
- `♂のみ`
- `♀のみ`

強制ペアでは、男女ペアを優先して作り、余った人同士で同性ペアを作ります。人数が奇数の場合は、1人だけ余りとして表示されます。

## スマホの渡し方

このアプリでは、画面に次の操作担当者が大きく表示されます。

- `次の入力者`
- `次の回答者`
- `次の投票者`
- `Private Turn`
- `Private Answer`
- `Private Vote`

これらが表示されたら、名前の人にスマホを渡します。本人が操作を終えたら、画面のボタンを押すと内容が隠れ、次の人へ渡す画面になります。

## データの扱い

- 入力したメンバー情報は、基本的にそのブラウザ内だけで使われます
- 秘密入力、投票内容、回答内容はサーバーには保存されません
- ゲーム途中でページを再読み込みすると、ゲームの途中状態は最初に戻ります
- ブラウザやタブを閉じた場合、メンバー設定がやり直しになることがあります

## うまく遊ぶコツ

- 最初に「パスあり」「無理強いなし」を全員で確認する
- 秘密画面では、周りの人は画面をのぞかない
- 入力が終わったら画面を伏せ気味にして次の人へ渡す
- 結果は笑って受け止める
- 微妙な空気になりそうなら、すぐ次のゲームへ行く
